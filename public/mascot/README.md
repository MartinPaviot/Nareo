# Aristo Mascot Images

Place the following mascot images in this directory:

1. **mascotte.png** - Default/listening state (graduation hat cat)
2. **Processing.png** - Reading/thinking state
3. **Talking.png** - Speaking/asking questions state
4. **Happy.png** - Correct answer/happy state
5. **Disappointed.png** - Confused/needs help state
6. **adcdebda.png** - Trophy/success state

## Image Specifications
- Format: PNG with transparency
- Recommended size: 256x256px or 512x512px
- Style: Friendly, cartoon-style graduation hat cat mascot
